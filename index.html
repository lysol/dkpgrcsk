douglbutt by lysol

IRC bot written in Python, using irclib

Douglbutt is the bot that runs in #mefi on irc.slashnet.org and posts to Twitter as @douglbutt.

Dependencies

pymblr
poster
elementtree
mechanize

Install

Copy douglbutt.conf-dist
Edit to your liking.
python douglbutt.py douglbutt.conf
